# MOVIES DATA DASHBOARD AND ANALYSIS IN MS EXCEL
This project focuses on analyzing a Bollywood movie dataset using Microsoft Excel to generate meaningful insights and visualize key trends in the film industry. The dashboard was designed to be interactive and user-friendly, enabling dynamic exploration of data across multiple dimensions.

The dataset was cleaned and structured to ensure consistency and accuracy. Advanced Excel features such as Pivot Tables, slicers, and charts were used to transform raw data into an interactive dashboard. The project analyzes factors such as movie ratings, genres, revenue, and overall performance to identify patterns and trends.

The dashboard allows users to filter data dynamically using slicers, making it easy to compare different categories and extract insights efficiently. It provides a clear visual representation of data, helping in understanding industry trends and performance metrics.
